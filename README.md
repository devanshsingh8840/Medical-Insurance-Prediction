# 🏥 Medical Insurance Cost Prediction

## 📌 Project Name

Medical Insurance Cost Prediction using Machine Learning

---

## 📖 Objective

The objective of this project is to predict the medical insurance charges of a person based on various factors such as age, gender, BMI, number of children, smoking habits, and region using Machine Learning. This project helps insurance companies estimate medical insurance costs quickly and accurately.

---

## 📊 Dataset

Dataset Name: Medical Cost Personal Dataset

Source:
https://www.kaggle.com/datasets/mirichoi0218/insurance

Dataset contains:

- Age
- Sex
- BMI
- Children
- Smoker
- Region
- Charges (Target Variable)

Number of Records: 1338

- R² Score

## ⚙ Installation

1. Clone the repository

2. Open the project folder

3. Install the required libraries

pip install -r requirements.txt

4. Run the application

python application.py

---

## ▶ How to Run

Open your browser and visit

http://127.0.0.1:5000

Enter:

- Age
- Gender
- BMI
- Number of Children
- Smoker Status
- Region

Click **Predict Insurance Cost** to get the estimated insurance charge.
